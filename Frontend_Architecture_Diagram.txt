ROUTING STRUCTURE:
==================

/ (LoginView)
│
└─ /dashboard (DashboardView)
   ├─ / (HomeView)
   ├─ /equipments (EquipmentView - admin only)
   │  └─ /:id (EquipmentDetailsView)
   │  └─ /:id/rental-history (EquipmentRentalHistoryView - admin only)
   │
   ├─ /customers (CustomerView - admin only)
   │  └─ /:id (CustomerDetailsView)
   │  └─ /:id/rentals (RentalView)
   │
   ├─ /rentals (RentalView - admin only)
   │  └─ /:id (RentalDetailsView - admin only)
   │  ├─ /issue (IssueEquipmentView)
   │  ├─ /return (ReturnEquipmentView)
   │  ├─ /extend (ExtendRentalView - admin only)
   │  └─ /cancel (CancelRentalView - admin only)

==========================

Store: Authentication
├─ State:
│  ├─ user info (from cookie)
│  ├─ authRole (Admin/User)
│  └─ token info
│
├─ Actions:
│  ├─ checkAspNetCoreCookie()
│  └─ logout()
│
└─ Persistence: Via pinia-plugin-persistedstate
