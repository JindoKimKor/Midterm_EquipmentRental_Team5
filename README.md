# Equipment Rental System - Setup Guide

## Setup Instructions

### Backend (API)

1. Navigate to backend directory:

```bash
cd Midterm_EquipmentRental_Team5
```

2. Run the API:

```bash
dotnet run
```

API will run at: http://localhost:5115

For Swagger Doc at: http://localhost:5115/swagger

### Frontend (UI)

1. Navigate to frontend directory:

```bash
cd Midterm_EquipmentRental_Team5_FrontEnd
```

2. Install dependencies:

```bash
npm install
```

3. Run the application:

```bash
npm run dev
```

UI will run at: http://localhost:5173
